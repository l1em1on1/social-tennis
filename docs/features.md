As a user I want to be able to:
- join multiple leagues (solo or with partner)
- see the list of my leagues 
- see the list of leagues that I can join 
- add the scores and date of the games I played 
- add substitute for the player of the played game
- add and edit my availability for the game during specific relegation phase
- plan the games date and allow to add it to the calendar 
- show court availability for the dates that all players of specific match have availability for
(https://stg-itf-docs.clubspark.io/api-docs/)

As a manager I want to be able to:
- create leagues (mens doubles, ladies doubles, mixed doubles, singles and foursomes)
- approve players to join league (for doubles I want users to be allowed joining as a pair or solo to match pairs)
- split users to groups of n teams.
- edit groups mid league in case someone wants to exit the league
- track scores of the games and keep stats
- set league duration
- set league relegation phase frequency
